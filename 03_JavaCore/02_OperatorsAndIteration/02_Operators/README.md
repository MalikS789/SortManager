### Binary Operators

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)

### Numeric Promotion
- If two values have different data types, the larger one is used for both types.
  
### Unary Operators
- !
- \+
- ++
- \--

### Local Operators 
- x & y
- x | y
- x ^ y
- x && y; if x is false, y is not checked
- x || y; if x is true , y is not checked
